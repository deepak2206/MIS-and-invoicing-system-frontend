MIS and Invoicing System


This is a simple authentication app built using **React.js**. It includes fully functional **Login** and **Register** pages, and can be used as a starter template for apps that require user authentication.

🧰 Tech Stack

- [React.js](https://reactjs.org/) (Vite)
- HTML5 + CSS3 (No Bootstrap/Tailwind)
- Axios (for HTTP requests)
- React Router DOM (for navigation)
- GitHub for version control

✨ Features

- ✅ User Registration (Name, Email, Password, Role)
- ✅ Login with Email and Password
- ✅ Error and success messages
- ✅ Responsive design
- ✅ CSS styled components (no external UI library)
- ✅ Clean and simple code structure
